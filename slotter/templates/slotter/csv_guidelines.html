Spreadsheet Guidelines

Each row in the spreadsheet should represent a student's data.

Biographical data

  • the first four columns (A-D) should contain data for CNetIDs, last names, first names, and pronouns in exactly that order
  • CNetIDs can contain alphanumeric characters but no spaces (up to 30 characters)
  • Last names and first names can contain alphabetical characters, hyphens, and spaces (up to 30 characters)
  • Pronoun specifications can contain alphabetical characters and forward slashes (up to 25 characters), e.g. they/them/theirs
  • headers for these columns can be named however you like, but they can only contain alphabetical characters and spaces

Scheduling Data

  • additional columns (E and onwards) should indicate a student's availability for a given timeslot
  • if a student is available for a time, indicate this with Available, and if not, indicate this with Not available
  • each header should define a timeslot by specifying a day of the week, start time, and end time in that order, e.g. Monday, 9:30 am - 10:50 am
    • the entry must start with an alphabetical character
    • the day of the week must be a weekday and must be identifiable by its first three letters or spelled out in full, e.g. Wed or Wednesday
    • times must be specified in 12-hour format and end with an am or pm
    • a hyphen must separate the start time from the end time
    • no leading zeros on times, e.g. 2:00 pm, not 02:00 pm
    • square brackets are admissible, e.g. Availability [Monday, 9:30 am - 10:50 am]
